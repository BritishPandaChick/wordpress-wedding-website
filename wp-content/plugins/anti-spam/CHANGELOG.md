####   Version 7.5.0 (2026-02-23)

### New Features

- Added advanced spam filtering based on Machine Learning (Pro)
- New modern dashboard with improved user interface and navigation
- Activity logger with detailed UI to track security events
- Compatibility with PHP 7.4 and above.

### Enhancements

- Migrates license management to the Themeisle store (Pro)
- Better text domain translations across PHP and JavaScript files.
- Improved the FTP and Dropbox connection in backup settings.
- Improved the overall working of Two Factor module.
- Removed the Hide Login plugin
- Removed deprecated features such as Site Checker, Firewall, Malware, Scanner modules from the plugin.

### Bug Fixes

- Fixed _load_textdomain_just_in_time error affecting translations
- Removed duplicate dashboard link in admin menu
- Fixed privacy policy notice display issues
- Corrected text domain in JavaScript files for proper translations
